## 1.3.0+mc1.21.11 Changelog

### ***CONFIGURATION***
- Configuration file has been moved to `\config\servertweaks\` directory. Previous configuration files won't be used anymore

### ***NEW CONTENT***
- `/discard <target>` command, allows operators discard entities, blocks, and clear inventories (of any entity/block)
  - Entity discard - basically `/kill` without dropping loot or XP, just poof, and it's gone
  - Block discard - basically `/setblock x y z minecraft:air` without specifying air and no items or anything else (like XP) drops from the block
  - Inventory discard - basically `/clear` mixed with `/data` but much less complex
- Right-Click-Harvest
  - Yes fortune applies
  - Larger affect area when using a hoe
- Special Player Abilities (UNFINISHED)
  - All abilities based on Minecraft mobs
  - Can be configured to be disabled
  - Player abilities can only be given by editing `player_abilities.json` file in `\config\servertweaks\` FOR NOW
    - A proper config add/remove for player abilities will be added once testing and experiments will be done
- `/fillExtras` command because vanilla didn't have what I needed
  - `/fill` but better
  - Modes
    - `replaceOnly` - Basically vanilla's `replace` I guess
    - `destroyOnly` - Destroys only specified block
    - `silkDestroy` - Destroys all blocks with silk touch effect
    - `silkDestroyOnly` - `destroyOnly` but with silk touch effect
    - `fortuneDestroy` - Destroys all blocks with fortune effect, fortune level ahs to be specified
    - `fortuneDestroyOnly` - `destroyOnly` but with fortune effect, fortune level ahs to be specified
- `/fly <player>` command, allows for creative flight in survival
  - Can also be executed standalone as a toggle (`/fly` with no selectors)

### ***REMOVED***
- Removed `/daycount` command
- Removed `/duel` command
<small>Both can be restored with a custom build of the mod, removed classes are in the mod's repository

### ***BUG FIXES***
- Fixed broken duel invites messages (Command feature was removed shortly after)

### ***TECHNICAL CHANGES***
- Removed "[ServerTweaks] " prefix from all messages sent by the mod

### ***DEV STUFF***
- Moved command classes from `net.justmili.servertweaks.commands` to `net.justmili.servertweaks.content.commands`
- Moved util classes from `net.justmili.servertweaks.util` to `net.justmili.servertweaks.core.util`
- Moved and renamed player data attachments class from `net.justmili.servertweaks.fdaapi` to `net.justmili.servertweaks.core.variables`
- Renamed package `net.justmili.servertweaks.init` to `net.justmili.servertweaks.registries`
- Renamed package `net.justmili.servertweaks.mechanics.events` to `net.justmili.servertweaks.mechanics.logic`
